# `type` / `not`

**Category:** Universal  
**Input type:** Any

## What's implemented

`InputType::Any` catalog entries. Both are zero-argument:
- `type` returns the JSON type name as a string (`"string"`, `"number"`, `"array"`, `"object"`, `"boolean"`, `"null"`)
- `not` negates a boolean

## What's missing

Nothing. Zero-argument transforms.
